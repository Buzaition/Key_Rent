# KEY Car Rental & Fleet Management System

A comprehensive, modern web application for car rental agencies. This platform provides an intuitive customer-facing portal along with a powerful administrative dashboard for managing the fleet, customers, contracts, invoices, and more.

## 🚀 Features

### For Customers
- **Sleek Landing Page:** Browse the fleet, view offers, and easily search for available vehicles.
- **Multi-language Support:** Seamlessly switch between English and Arabic.
- **Modern UI:** Engaging animations, beautiful dark mode, and responsive design tailored for any device.

### For Administrators
- **Comprehensive Dashboard:** Monitor active rentals, upcoming maintenance, and revenue.
- **Fleet Management:** Track vehicle statuses (Available, Rented, Maintenance, Out of Service), track mileage, and log damages.
- **Customer Database:** Manage individual and corporate clients, including their documents and outstanding balances.
- **Contract & Invoice Engine:** Generate automated rental agreements, track payments, and export detailed financial reports.

## 🛠️ Technology Stack

- **Frontend Framework:** React 18 with Vite
- **Styling:** Tailwind CSS & Lucide React Icons
- **State Management:** Zustand
- **Routing:** React Router DOM
- **Internationalization:** i18next (English & Arabic)
- **Forms & Validation:** React Hook Form
- **Animations:** Framer Motion
- **Data Persistence:** LocalStorage Mock DB

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Buzaition/Key_Rent.git
cd Key_Rent
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
